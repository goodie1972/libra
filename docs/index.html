🎯设计哲学

一页一主角
每页只有一个视觉重心。K 线页主角是图表，不是侧栏不是标题。
数据优先
装饰服务于数据。不用渐变卡片、不用发光按钮来吸引注意。
间距即语言
4px 为基数。相关元素间距小，不相关元素间距大。
减法制造张力
层次靠明度差和间距。100 个组件不如 3 个精准的间距。
风格锚定：Linear（暗色体系）+ Stripe（金融排版）+ Bloomberg Terminal（金融数据基因）

📦快速开始

# 在你的项目中安装
npm install @libra/tokens

# 在入口文件引入
import '@libra/tokens/css'
# 安装主题引擎（浏览器环境）
npm install @libra/theme

import { applyPreset } from '@libra/theme'
applyPreset('dark'); // 🌙 暗色
applyPreset('soft'); // 🌤️ 柔光（默认）
applyPreset('light'); // ☀️ 亮色
# 用 CLI 安装组件（类似 shadcn）
npx @libra/cli init
npx @libra/cli add button
npx @libra/cli add card
npx @libra/cli add table
